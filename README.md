## 📋 Topics 

```

📦 SQL
│-- 001-WALKTHROUGH.sql        # A quick overview of SQL concepts
│-- 002-DB_CREATION.sql        # Create database and basic setup

│-- 010-FULL_JOIN.sql          # FULL JOIN (using UNION)
│-- 011-CROSS_JOIN.sql         # Cartesian product (CROSS JOIN)
```
